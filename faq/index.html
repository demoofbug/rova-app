常见问题 (FAQ)

在这里找到您关于嗅影播放器最常见问题的答案。

[常规] 嗅影播放器是免费的吗？

是的，嗅影播放器目前完全免费。我们致力于为用户提供一个纯净、高效的视频嗅探和播放体验。

[常规] 软件支持哪些操作系统？

目前我们主要支持 Windows 和 macOS 平台，Android、iOS版本也已提供。未来我们计划支持Web平台。

[功能] 嗅影支持平板安装吗？

支持的，嗅影兼容 Android 平板和 iPad，用户可以在这些设备上正常安装和使用。

[功能] 视频嗅探失败了怎么办？

点击刷新按钮，重新加载网页，会重新触发嗅探逻辑

[IPA签名] IPA签名工具使用教程

推荐使用爱思助手、AltStore 或 Sideloadly 进行自签名。点击查看爱思助手AppleID自签名教程

[IPA签名] 是否必须用电脑连接手机？

大部分自签名工具（如 AltStore、Sideloadly、爱思助手）需要用电脑连接手机完成安装。无电脑在线签名虽然方便，但稳定性和安全性较低，证书容易失效。

[IPA签名] 签名时是否需要 Apple ID？

自签名通常需要一个 Apple ID 来生成临时证书和描述文件。建议使用专门的小号 Apple ID，避免使用主号以减少风险。

[IPA签名] 自签名 IPA 有效期多久？

免费 Apple ID 的签名有效期为 7 天，到期后需要重新签名安装；付费开发者账号的签名有效期为 1 年，到期前需要重新签名。

[IPA签名] 一个 Apple ID 签名的 IPA 是否能安装在多台设备？

免费 Apple ID 通过自签名生成的 IPA 只能安装在绑定签名时使用的那一台设备上。每次签名时，签名工具会生成与设备 UDID 绑定的描述文件，因此同一 IPA 无法直接跨设备安装，除非重新使用目标设备进行签名。

[IPA签名] 签名后是否需要信任？

使用企业证书签名的 IPA 需要在“设置 → 通用 → VPN与设备管理”中手动信任；免费 Apple ID 自签名也是相同的流程。
